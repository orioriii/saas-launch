# Cloudflare の登録とログイン

バックエンド（Workers）のデプロイ先です。**無料枠**で動き、クレジットカードは不要です。

## 1. アカウントを作る

1. ブラウザで <https://dash.cloudflare.com/sign-up> を開く
2. **メールアドレス**と**パスワード**を入力して「Sign Up」をクリック
3. 登録したメールアドレスに**確認メール**が届くので、本文のリンクをクリックして認証
4. これで Cloudflare のダッシュボードにログインできる状態になります

> すでにアカウントを持っている場合は、この手順は不要です。

## 2. CLI（wrangler）でログインする

ハーネスは Cloudflare の公式ツール **wrangler** を使ってデプロイします。
`saas-launch setup` を実行すると、次を自動で行います（手動でも実行できます）:

```bash
npx wrangler login
```

- 実行するとブラウザが開きます。
- 「**Allow**（許可）」ボタンを押すと、ターミナルにログイン完了が表示されます。
- ブラウザが自動で開かない場合は、表示された URL を手動で開いてください。

## 3. ログインできたか確認する

```bash
npx wrangler whoami
```

- メールアドレスと **Account ID**（32桁の英数字）が表示されれば成功です。
- ハーネスはこの Account ID を自動で控えます。

## うまくいかないとき

| 症状 | 対処 |
|------|------|
| `wrangler` が見つからない | Node.js がインストールされているか確認。`npx wrangler ...` の形で実行 |
| ブラウザが開かない | 表示された URL を手動でコピーしてブラウザに貼り付け |
| ログインしたのに未ログイン扱い | `npx wrangler logout` してから再度 `npx wrangler login` |
| 会社ネットワークで弾かれる | 別のネットワーク（テザリング等）で試す |

ログインができたら、`saas-launch setup` に戻って続きから進めてください。
